Coursework

University of Pittsburgh - School of Computing and Information

Sep 2019 - Dec 2023




B.S. Computer Science, Minor in Economics

Software Engineering




Explored modern software engineering best practices. Topics covered include common software architectural patterns, design patterns, and development processes, with focus on an Agile Scrum group project.

Programming Languages for Web Applications




Discussed numerous languages and their functions in the development of web apps. Topics included HTML/CSS, Javascript, Python (Flask), Rust, WASM, basic databases, and more.

Data Structures & Algorithms




Explored methodology and implementation of integral data structures and common algorithms. Over two courses, topics covered include sorting and searching algorithms, runtime, numerous data structures, dynamic programming, and more.

Introduction to Operating Systems




Learned about the basic components of operating systems and UNIX like kernels. Topics covered include synchronization problems, CPU scheduling, virtual memory, file systems, and more.

Systems Software




Learned about the basic constructs of computer software as a system. Topics covered include data representation, C language, x86 assembly, memory management, virtual memory, caching, and more.

Computer Organization & Assembly Language




Studied concepts related to the design and functionality of computer hardware. This course introduced many important topics such as registers, memory, and MARS assembly.

Formal Methods




Explored theoretical computer science and the logic behind it. Covered topics such as computability, decidability, and reducability.

Composing Digital Media




Created digital projects of different mediums to drive rhetoric. Learned about audio and image editing software, as well as HTML and CSS.
